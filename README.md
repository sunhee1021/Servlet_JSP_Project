# JYP(JEJU Yeohang Planner)
JYP(JEJU Yeohang Planner)는 약 2주간 진행하여 만든 2차 프로젝트입니다.  (기간 : 16일 , 인원 : 5명)<br>
JAVA와 JavaScript, JSP 만을 이용하여 MVC패턴의 Model 2 방식으로 진행하였습니다.<br>

<ul><li>'어스토리'라는 사이트를 벤치마킹하여 제작하였습니다.</li></ul>
